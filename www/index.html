          This webapp plots the 
          iris dataset
          bundled with R language distribution.
        

        

          You can choose the X and Y variable and the
          plotting on the left will be updated.
        

        

          After plotting the desired variables, the server will calculate
          a linear regression model for the desired variables (with Y 
          variable as the predicted value) and draw the line of the 
          resulting linear model in the graph. The summary of the linear regression
          model is also dumped below the graph.
